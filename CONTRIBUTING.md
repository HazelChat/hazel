# Contributing

Thanks for your interest in contributing to Hazel Chat!

## Development Setup

See [README.md](./README.md) for instructions on setting up your local development environment.

## Pull Requests

1. Fork the repository and create your branch from `main`
2. Make your changes
3. Run `bun run format` to format code
4. Run `bun run test:once` to ensure tests pass
5. Open a pull request

## Code Style

This project uses [OXC](https://oxc.rs/) for formatting and linting:

- Tab indentation
- Double quotes
- Trailing commas
- 110 character line width
