## Accordion

Create collapsible FAQ sections:

<Accordions type="single">
  <Accordion title="How do I get a bot token?">
    Navigate to **Settings > Integrations** in the Hazel dashboard and click "Create Bot" to generate a new token.
  </Accordion>
  <Accordion title="Can I run multiple bots?">
    Yes! Each bot requires its own token. You can create as many bots as needed from the integrations page.
  </Accordion>
  <Accordion title="What events can my bot listen to?">
    Bots can listen to messages, reactions, channel joins/leaves, and more. See the [Events Reference](/bot-sdk/reference/events) for the full list.
  </Accordion>

## Code Blocks

Syntax-highlighted code with copy button:

```typescript title="bot.ts"
import { HazelBot } from '@hazel/bot-sdk'

const bot = new HazelBot({
  token: process.env.BOT_TOKEN!,
  debug: true,
})

bot.on('message', async (message) => {
  if (message.content === '!ping') {
    await message.reply('Pong!')
  }
})

bot.start()
```

### Line Highlighting

```typescript
import { HazelBot } from '@hazel/bot-sdk' // [!code highlight]

const bot = new HazelBot({
  token: process.env.BOT_TOKEN!,
})

bot.on('message', async (message) => { // [!code highlight]
  console.log(message.content)
})
```
